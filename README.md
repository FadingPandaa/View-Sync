## 📖 Project Overview

Visual Toggle is a Kotlin Android application focused on demonstrating how image visibility can be controlled dynamically within a mobile application.

The project showcases:

* Kotlin programming fundamentals
* Android UI interaction
* Visibility control using user actions
* Event handling and dynamic interface updates

------------------
## 🚀 Features

✅ Show and hide images dynamically

✅ Interactive user interface

✅ Button click event handling

✅ Structured Kotlin application design

✅ Simple and responsive functionality

------------------------
## 🛠️ Technologies Used

* Kotlin
* Android Studio
* Android SDK
* Gradle

-----------------
## ▶️ How to Run

Using Android Studio:

1. Open the project in Android Studio
2. Allow Gradle dependencies to sync
3. Build the project
4. Run the application using an emulator or Android device
                  
-------------------
## 🔄 How It Works

1. The application launches and displays an image
2. The user interacts with buttons or controls
3. The image visibility changes dynamically:
   * Visible
   * Hidden
4. The UI updates instantly based on user interaction
